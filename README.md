# AR Succulent Garden

This is a mobile AR app made using Unity and Firebase where users can grow virtual succulents. Each plant needs to be watered and fertilised before it becomes fully grown. Users can learn more about succulents.

**What the app does**

- Login / Register using Firebase
- Scan image cards to spawn plants in AR
- Plant seeds into pots
- Water and fertilise plants to help them grow
- Fully grown plants can be rotated and inspected
- Plant progress is saved to Firebase
- End screen appears after all plants are matured
- Web dashboard to manage plant info and view user progress

**How to use**

- Open the app on a mobile device
- Login or register an account
- Enter the garden
- Scan a seed card
- Plant the seed and care for it
- Inspect the plant once it is fully grown
- After all plants are grown, the experience ends

**Web dashboard**

The web dashboard is used to:

- Add or edit plant species information
- Add reminders for each plant
- View user plant progress stored in Firebase
- This dashboard connects to the same Firebase project as the Unity app.

**Tools used**

- Unity (AR Foundation, XR Interaction Toolkit)
- Firebase Authentication
- Firebase Realtime Database
- HTML, CSS, JavaScript, C#

**Notes / limitations**

- Firebase works on Android only
- Desktop testing may not support Firebase properly
- Dashboard is for admin use only

**Assets**

- Free plant models from Sketchfab
- ChatGPT used for guidance purposes only
